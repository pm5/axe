Usage
-----

For each level:

```
$ cd <level_dir>
$ npm i && node xit.js          # Run Node.js
$ go run xit.go                 # Run Go
$ cabal install && cabal run    # Run Haskell
```

All examples save output to `output.json` in the level directory.
